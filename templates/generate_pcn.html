How to Generate a Label
  1. Enter the required Part Number (job number)
  2. Add optional fields like Order Reference, Supplier PN, Quantity, Date Code
  3. Select sensitivity level if applicable
  4. Click "Generate Lot Number & Print Label"
  5. System auto-assigns next 5-digit lot number
  6. Print dialog opens automatically - select your printer and print!
Important Notes
  • Lot numbers are auto-incremented starting from 00001
  • Each lot number is unique and cannot be reused
  • Labels include CODE128 barcode for scanner compatibility
